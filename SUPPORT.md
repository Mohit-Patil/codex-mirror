# Support

## Usage help

- Read `README.md` first.
- Check `docs/TROUBLESHOOTING.md` for common fixes.

## Bug reports

Open a GitHub issue with:

- OS and terminal
- Node.js version
- codex version (`codex -V`)
- exact command / TUI flow
- expected vs actual behavior

## Feature requests

Open a GitHub issue with:

- problem statement
- proposed behavior
- alternatives considered
